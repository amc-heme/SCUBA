# Get default reduction from object

Returns the default slot for the object passed. This function was
deprecated in version 0.10.0. Please use
[`default_layer()`](https://amc-heme.github.io/SCUBA/reference/default_layer.md)
instead.

## Usage

``` r
default_slot(object)
```

## Arguments

- object:

  a single cell object supported by SCUBA. Currently, Seurat,
  SingleCellExperiment, and anndata objects are supported.
