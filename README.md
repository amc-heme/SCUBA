# SCUBA

###  *S*ingle *C*ell *U*nified *B*ack end *A*PI 

SCUBA is a unified data accession interface for single cell data formats. The package pulls data from Seurat, SingleCellExeriment, and Anndata objects for use in downsteam plotting and analysis. Plotting functions that produce Seruat-style plots from all three object formats are also included.
